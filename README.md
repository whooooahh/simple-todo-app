# Simple Todo App

Git チーム開発学習用のシンプルなTodoアプリケーションです。

## 機能

- [ ] タスクの追加
- [ ] タスクの完了/未完了切り替え  
- [ ] タスクの削除
- [ ] タスクの編集（今後実装予定）
- [ ] タスクの検索・フィルター（今後実装予定）

## 技術スタック

- HTML5
- CSS3
- Vanilla JavaScript

## 開発環境

- エディタ: Cursor
- バージョン管理: Git & GitHub
